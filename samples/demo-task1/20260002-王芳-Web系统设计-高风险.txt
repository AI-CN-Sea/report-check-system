实验报告题目：Web系统设计与实现

本次实验完成了一个基于 Spring Boot 和 Vue 的实验报告查重系统原型。系统采用前后端分离架构，后端负责用户认证、课程管理、实验任务管理、报告上传和查重计算，前端负责页面展示、表单提交和统计图表渲染。数据库使用 MySQL 保存用户、班级、课程、任务、报告文件和查重结果等信息。

在需求分析阶段，系统主要面向管理员、教师和学生三类用户。管理员维护基础数据，教师创建实验任务、上传或查看学生报告、发起查重任务并查看结果，学生查看任务并提交自己的实验报告。这样的设计能够覆盖高校实验教学中常见的报告提交和批量检查流程。

在实现过程中，我重点完成了登录认证、文件上传、文本解析和结果展示模块。报告上传后，系统会读取 txt、docx 或 pdf 中的文本内容，并记录解析状态和字数信息。教师发起查重后，系统对同一实验任务下的报告进行两两比较，计算综合相似度并给出风险等级。

通过本次实验，我理解了 Web 系统从需求分析、数据库设计、接口开发到前端页面实现的完整流程，也认识到系统测试和异常处理对于项目稳定运行的重要性。
