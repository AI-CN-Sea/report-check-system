实验报告题目：Web系统设计与实现

本次实验围绕校园二手书交易平台展开设计。系统主要包括用户注册登录、图书发布、图书搜索、订单管理和留言沟通等功能。后端使用 Spring Boot 提供 REST 接口，前端使用 Vue 构建页面，数据库使用 MySQL 保存图书、用户和订单数据。

在需求分析中，我将用户分为普通学生和管理员。学生可以发布闲置教材、浏览图书信息、提交购买意向，管理员负责处理违规信息和维护分类数据。系统设计时重点考虑了图书信息的检索效率和用户操作流程的简洁性。

实现过程中，我完成了数据库表结构设计、接口参数设计和部分页面原型。通过测试发现，文件上传、输入校验和异常提示会明显影响用户体验，因此系统需要在后续开发中继续完善提示信息和错误处理逻辑。

通过本次实验，我熟悉了 Spring Boot 与 Vue 的协同开发方式，并对 Web 应用的分层结构有了更加清晰的认识。
