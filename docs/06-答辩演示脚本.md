# 答辩演示脚本

## 1. 项目介绍

本系统面向高校实验教学场景，用于辅助教师对学生提交的实验报告进行批量查重和相似度分析。

## 2. 演示流程

1. 打开系统登录页。
2. 使用管理员账号登录，演示用户管理、课程管理、班级管理和班级学生分配。
3. 退出后使用教师账号登录。
4. 查看首页数据看板。
5. 创建或查看实验任务。
6. 单文件上传或批量上传学生实验报告。
7. 发起查重任务。
8. 查看查重结果、风险等级筛选和相似句子。
9. 导出查重结果 CSV。
10. 查看统计分析图表。

## 2.1 可直接演示的数据

系统已提供两份样例报告：

```text
samples/student01-report.txt
samples/student02-report.txt
```

演示时可以将它们分别上传给学生“李明”和“王芳”，然后发起查重。系统会生成一条中风险查重结果。

## 3. 技术说明重点

- 后端采用 Spring Boot 分层架构。
- 数据库采用 MySQL，围绕用户、课程、任务、报告、查重结果建模。
- 查重算法采用 TF-IDF、余弦相似度和 SimHash。
- 前端采用 Vue 3 和 Element Plus，实现管理端页面。
