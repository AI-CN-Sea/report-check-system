# 数据库说明

数据库名称：

```text
report_check_system
```

初始化脚本：

```text
database/init.sql
```

可选演示数据脚本：

```text
database/demo_seed.sql
```

如果希望进入系统后立即看到查重结果和相似句详情，可在 `init.sql` 后执行 `demo_seed.sql`。如果希望展示真实上传流程，则不执行该脚本，直接使用 `samples/generated` 下的样例报告批量上传。

## 核心表

| 表名 | 说明 |
| --- | --- |
| sys_role | 系统角色表 |
| sys_user | 系统用户表 |
| course | 课程表 |
| class_info | 班级表 |
| student_class | 学生班级关联表 |
| experiment_task | 实验任务表 |
| report_file | 实验报告表 |
| check_task | 查重任务表 |
| check_result | 查重结果表 |
| similar_sentence | 相似句子表 |
| operation_log | 操作日志表 |
