# 演示数据与查重场景说明

## 1. 初始化数据规模

执行 `database/init.sql` 后，系统会初始化以下演示数据：

```text
用户：1 个管理员、3 个教师、12 个学生
班级：3 个班级
课程：3 门课程
实验任务：3 个主演示任务
```

默认账号如下：

| 用户类型 | 用户名 | 密码 | 说明 |
| --- | --- | --- | --- |
| 管理员 | admin | 123456 | 管理用户、课程、班级 |
| 教师 | teacher01 | 123456 | 管理计算机综合课程设计 |
| 教师 | teacher02 | 123456 | 管理 Java Web 应用开发 |
| 教师 | teacher03 | 123456 | 管理软件工程实践 |
| 学生 | student01 至 student12 | 123456 | 用于多格式报告上传与查重演示 |

## 2. 多格式样例报告

系统提供真实可打开的 txt、docx、pdf 三类样例报告：

```text
samples/generated/task1-txt/    12 份 txt 报告
samples/generated/task2-docx/   12 份真实 Word 报告
samples/generated/task3-pdf/    12 份文字型 PDF 报告
```

文件名均包含学号和姓名，教师端批量上传时可以自动匹配学生。

## 3. 查重风险案例

每个主演示任务都包含高风险、中风险、低风险和正常四类案例：

| 案例 | 学生组合 | 说明 |
| --- | --- | --- |
| 高风险 | 李明 / 王芳 | 多个自然句完全一致或高度相似 |
| 中风险 | 赵强 / 陈晨 | 部分公共句保留，部分内容改写 |
| 低风险 | 孙悦 / 周杰 | 只有少量设计思路相近 |
| 正常 | 吴敏 / 郑磊等 | 主题相关但表达角度明显不同 |

高风险和中风险样例中预置了可匹配的相似自然句，用于验证 `similar_sentence` 详情展示。

## 4. 可选预置结果

如果希望初始化后直接展示查重结果，可在 `init.sql` 后执行：

```text
database/demo_seed.sql
```

该脚本会插入：

```text
36 条 report_file 演示报告记录
3 条 check_task 查重任务
12 条 check_result 风险结果
多条 similar_sentence 相似句详情
```

如果希望展示真实流程，则不要执行 `demo_seed.sql`，按页面流程批量上传 `samples/generated` 目录下的报告后再发起查重。

## 5. 建议演示流程

```text
1. 管理员登录，查看 3 名教师、3 个班级、12 名学生。
2. 教师登录，进入报告上传页面。
3. 分别选择实验一、实验二、实验三，批量上传 txt、docx、pdf 样例。
4. 查看报告列表中的解析状态、字数和句子数。
5. 发起查重，查看查重任务列表和结果列表。
6. 打开高风险或中风险详情，查看相似句子。
7. 切换风险等级筛选，展示高/中/低/正常四类结果。
8. 查看统计分析图表并导出 CSV。
9. 尝试删除已关联课程、班级、任务或报告，验证中文失败原因提示。
```
